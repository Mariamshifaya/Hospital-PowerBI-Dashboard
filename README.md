# Hospital Management Dashboard (Power BI)

This project presents an interactive hospital analytics dashboard built using Power BI.

## Key Insights
- Total patients and appointments
- Doctor performance analysis
- Treatment cost breakdown
- Hospital billing and revenue trends

## Tools Used
- Power BI
- Excel
- Data Visualization

## Files Included
- Hospital_Dashboard.pbix
- Doctors.xlsx
- Patients.xlsx
- Appointments.xlsx
- Treatments.xlsx
- Billing.xlsx
